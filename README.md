Couple of simple scripts to make life easier when installing Apache24.
Instructions:

1. download this repository
2. download from Apache Lounge for example this file [httpd-2.4.66-260223-Win64-VS18.zip](https://www.apachelounge.com/download/VS18/binaries/httpd-2.4.66-260223-Win64-VS18.zip)
3. unzip Apache24 folder into same directory where these two scripts has been placed
4. run install_apache24.bat - thats all
5. uninstall and cleanup - uninstall_apache24.bat

Good luck!
